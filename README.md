Project Title: Mining Equipment Predictive Maintenance Analysis.

Objective: To identify the primary causes of mechanical failure in mining telemetry data.

Insights Found:
High Torque Correlation: Failed machines average 50.17 Nm vs 39.63 Nm for healthy units.

Primary Failure Mode: Heat Dissipation (112 counts).

Recommendations: Implement torque limits and improve cooling protocols during high-temp cycles.
